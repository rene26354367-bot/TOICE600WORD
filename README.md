# 多益 600 單字練習 Web App

手機優先的多益單字練習應用程式，日式手寫簡潔風格設計。
純靜態網頁，無需伺服器，直接用瀏覽器開啟即可使用。

---

## 專案結構

```
TOICE600WORD/
├── index.html   — 主體結構、分頁導航、Modal 彈出視窗
├── style.css    — 日式手寫簡潔風格樣式
├── data.js      — 單字資料（目前 5 個範例，請自行填充至 600 個）
└── app.js       — 所有應用程式邏輯
```

---

## 使用方式

直接用瀏覽器開啟 `index.html`，無需安裝任何套件或啟動伺服器。

---

## 功能說明

### 單字列表
- 顯示所有單字卡片（單字 + 音標）
- 已背過的單字卡片左側有朱紅邊線與「已背過」徽章
- 點擊卡片 → 開啟詳細 Modal（定義 + 例句 + 切換按鈕）

### 進度條
- 頂部即時顯示：已背過 X / N（百分比）
- 每次切換「已背過」狀態後立即更新

### 測驗
- 從「未背過」的單字中隨機抽取最多 10 題
- 四選一選擇題（題目為英文單字，選項為中文定義）
- 點擊選項後鎖定，顯示答對（綠色）/ 答錯（紅色），1 秒後自動換題
- 測驗結束後顯示得分與「再挑戰」按鈕
- 未背過單字少於 4 個時顯示提示（四選一最少需要 4 個不同定義）

### 進度儲存
- 使用 `localStorage` 儲存已背過狀態
- 重新整理或關閉瀏覽器後，進度不會消失
- 之後新增至 `data.js` 的單字，預設皆為「未背過」

---

## 新增單字（填充 600 個）

打開 `data.js`，在陣列末端繼續新增，格式如下：

```js
{
  id: 6,
  word: "authorize",
  phonetic: "/ˈɔːθəraɪz/",
  definition: "授權；批准",
  example: "Only the manager is authorized to approve expenses.",
  mastered: false
},
```

**注意事項：**
- `id` 必須唯一，建議依序遞增
- `mastered` 固定填 `false`，程式會自動從 `localStorage` 讀取實際狀態
- 每筆資料後面加逗號（最後一筆除外）
- 新增完成後，直接重新整理網頁即可看到新單字

---

## 單字資料欄位說明

| 欄位 | 型別 | 說明 |
|------|------|------|
| `id` | Number | 唯一編號（建議 1, 2, 3… 依序遞增）|
| `word` | String | 英文單字 |
| `phonetic` | String | 音標（KK 或 IPA 皆可）|
| `definition` | String | 中文定義 |
| `example` | String | 英文例句 |
| `mastered` | Boolean | 固定填 `false`（由程式管理）|

---

## 設計風格

- **色調**：米白底色（`#faf8f3`）、墨黑文字、朱紅強調色
- **字型**：Noto Serif JP（標題）+ Noto Sans JP（內文）
- **卡片**：細線框線、充足留白，不使用陰影
- **按鈕**：邊框式墨印風格
- **進度條**：纖細低飽和，日式節制美感
- **需要網路**：字型透過 Google Fonts 載入；離線環境下會退為系統字型，功能不受影響

---

## localStorage 規格

| 項目 | 內容 |
|------|------|
| 鍵值 | `toeic_mastered` |
| 格式 | `{ "1": true, "5": true }` |
| 說明 | 只儲存已背過的單字 id；未出現的 id 視為未背過 |

---

## 清除所有進度

在瀏覽器開發人員工具的 Console 中執行：

```js
localStorage.removeItem('toeic_mastered');
location.reload();
```

或直接清除該網站的 localStorage。
